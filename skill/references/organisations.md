# Target Organisations

The full list of organisations Sam wants featured in the "Opportunities Newsletter" newsletter, structured in three tiers. **Tier 1** is the core monitoring loop — check every newsletter. **Tier 2** is active but lower-frequency, check every 2–3 newsletters. **Tier 3** is the Atlas Network directory used as a master source for anything not individually listed.

The newsletter does not include closed programs. The spreadsheet attachment also only includes currently-open programs.

## Organisations Sam has explicitly not asked to feature

- **Heritage Foundation** — explicitly excluded by Sam. Do NOT include in any newsletter or spreadsheet, even if their Young Leaders Program or any other Heritage offering is currently open.
- **Religious-conservative / Heritage-adjacent orgs** — explicitly excluded. Sam's audience is classical liberal / libertarian / centre-right and these orgs sit awkwardly with that readership.
- **Christian-right student orgs** — including Family Research Council, Discovery Institute, Becket Fund for Religious Liberty, and similar theologically-driven conservative orgs. Explicitly excluded.
- **Atlas Network Liberty Forums** as program listings — these are paid registration conferences, not student programs. Mention only if there's a specific student scholarship for an upcoming forum.
- **Students for Liberty** — Sam runs the ANZ region, so SFL gets dedicated emails rather than appearing in the regular roundup.
