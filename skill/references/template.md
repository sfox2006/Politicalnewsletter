# Email Template

The exact format Sam uses for "Opportunities Newsletter" editions.

## Subject line

```
Opportunities Newsletter
```

For single-program pitches (e.g. Young Voices solo, SFL solo), the subject can be more specific, e.g. "Opportunities Newsletter — SFL Local Coordinator Program".

## Opening block

For regular newsletters (most readers familiar with the list):

```
Hi!

Here's this week's roundup of programs currently open for students who lean classical liberal, libertarian, conservative, or centre-right. I've grouped them by country (Australia first), and within each country ordered by internship → student conference → student program/fellowship → online courses.

There is also a filterable spreadsheet of every currently-open program is attached — sort by country, type, deadline, international eligibility, etc.

Want to keep getting these? Sign up yourself or refer your friends who might find this beneficial: https://forms.gle/CStdPYuLTdF9RdYd8
```

For newsletters going to a freshly expanded list (when many new readers have been added — Sam will flag this):

```
Hi!

For those of you new to this list — welcome! "Opportunities Newsletter" is a roundup I send out every few weeks of open internships, fellowships, conferences, student programs, and online courses at think tanks, research centres, and educational foundations that lean classical liberal, libertarian, conservative, or centre-right. The goal is simple: surface the best opportunities in the space, so you don't have to track them down yourself.

If you'd rather not be on this list, just hit reply and let me know — no hard feelings, I'll take you off right away.

Want to keep getting these? Sign up yourself or refer a friend to help us grow: https://forms.gle/CStdPYuLTdF9RdYd8

Here's this week's roundup. I've grouped them by country (Australia first), and within each country ordered by internship → student conference → student program/fellowship → online courses.

📎 A filterable spreadsheet of every currently-open program is attached — sort by country, type, deadline, international eligibility, and more.
```

## Country sections

Country headers go on their own line, with a flag emoji, a space, and the country name in ALL CAPS. No markdown.

Standard sequence (skip any country with no currently-open programs):

```
🇦🇺 AUSTRALIA
🇳🇿 NEW ZEALAND
🇬🇧 UNITED KINGDOM
🇪🇺 EUROPE
🇺🇸 UNITED STATES
🇨🇦 CANADA
🌏 ASIA
🌎 LATIN AMERICA
🌍 MIDDLE EAST & AFRICA
🌐 ONLINE COURSES (free, global access)
```

Sections are separated by a horizontal rule of three hyphens on its own line:

```
---
```

## Program entries

Each program entry has three parts:

1. **Header line**: `Organisation Name — Program Name | Status or Deadline`
2. **Description paragraph**: 3–5 sentences covering what the program is, dates, cost/stipend, eligibility quirks, how to apply.
3. **URL line(s)**: bare URLs on their own line(s), no markdown link syntax.

Example:

```
Heritage Foundation — Young Leaders Program (Fall 2026) | Deadline: 31 May 2026

A 12–15 week paid internship in Washington, DC, paid at $18.50/hour. Includes the Boyce-Haller First Principles Lecture Series, weekly briefings on policy issues, professional development workshops, and access to Heritage events. International students authorised to work in the US are eligible. Rolling hiring — apply now.

https://www.heritage.org/young-leaders-program
```

## Ordering rules

Within each country, programs are ordered by type:
1. Internship (rolling first, then dated)
2. Student conference
3. Student program / fellowship / summer school
4. Essay competition / online course

When multiple internships exist from the same organisation, group them together (e.g. all three IEA internships in sequence).

## Closing

```
As always, check the links for full eligibility requirements, and the attached spreadsheet for a quick filterable overview. If you have any questions, feel free to reach out.

Best,
Sam
```

**Do NOT include a full signature block** — Gmail auto-appends Sam's signature.

## Formatting notes

- **No markdown asterisks or bold** in the body — Gmail's plain-text composer renders them as literal asterisks. Sam will manually bold/resize in the Gmail toolbar before sending.
- **No emojis** beyond the country flags and one section divider emoji per section.
- **Don't use the word "genuinely"** — Sam has flagged it as overused.
- **Don't use em-dashes excessively** — use sparingly for emphasis only.
- Date format: `31 May 2026` (day month year), not US-style.
- Currency: use the program's native currency. USD for US programs, GBP/£ for UK, AUD for Australian.

## Honest hedging language

When a deadline can't be verified from the live page:
- "Rolling" — for programs that genuinely accept rolling applications
- "Apply early" — when no deadline is shown but the program is dated
- "Contact for current application window" — when the program runs in cycles but the next cycle isn't yet posted
- "Applications open" — when the page confirms applications are accepting but no deadline is given

Never invent a specific deadline. Better to hedge honestly.
